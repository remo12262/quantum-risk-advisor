# Quantum Risk Advisor

AI-powered quantum risk assessment tool for organizations.

Evaluates post-quantum cryptography readiness and generates 
a personalized executive report with NIST FIPS 203/204/205 
gap analysis and migration roadmap.

## Features
- 4-step guided assessment
- Risk score 0-100
- AI-generated executive report
- NIST post-quantum standards compliance
- NIS2 and ACN alignment
